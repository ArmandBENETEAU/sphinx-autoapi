

ActionMethodSelectorAttribute Class
===================================



.. contents:: 
   :local:



Summary
-------

Base class for attributes which can implement conditional logic to enable or disable an action
for a given request. See [Microsoft.AspNet.Mvc.IActionConstraint]().
