

FileResult Class
================



.. contents:: 
   :local:



Summary
-------

Represents an [Microsoft.AspNet.Mvc.ActionResult]() that when executed will
write a file as the response.





Inheritance Hierarchy
---------------------
* :dn:cls:`System.Object`
* :dn:cls:`Microsoft.AspNet.Mvc.ActionResult`
