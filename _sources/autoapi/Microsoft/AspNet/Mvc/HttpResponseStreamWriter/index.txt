

HttpResponseStreamWriter Class
==============================



.. contents:: 
   :local:



Summary
-------

Writes to the [System.IO.Stream]() using the supplied [Microsoft.AspNet.Mvc.HttpResponseStreamWriter.Encoding](Microsoft.AspNet.Mvc.HttpResponseStreamWriter.yml).
It does not write the BOM and also does not close the stream.
