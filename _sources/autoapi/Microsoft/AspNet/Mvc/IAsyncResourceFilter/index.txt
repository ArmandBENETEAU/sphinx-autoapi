

IAsyncResourceFilter Interface
==============================



.. contents:: 
   :local:



Summary
-------

A filter which surrounds execution of model binding, the action (and filters) and the action result
(and filters).









Syntax
------

.. code-block:: csharp

   public interface IAsyncResourceFilter : IFilter
