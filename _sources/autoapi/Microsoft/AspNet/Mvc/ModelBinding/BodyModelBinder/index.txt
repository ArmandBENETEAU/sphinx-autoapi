

BodyModelBinder Class
=====================



.. contents::

Summary
-------

An [Microsoft.AspNet.Mvc.ModelBinding.IModelBinder]() which binds models from the request body using an [Microsoft.AspNet.Mvc.IInputFormatter]()
when a model has the binding source [Microsoft.AspNet.Mvc.ModelBinding.BindingSource.Body]()/
