

DataAnnotationsModelValidatorProvider Class
===========================================



.. contents:: 
   :local:



Summary
-------

An implementation of [Microsoft.AspNet.Mvc.ModelBinding.Validation.IModelValidatorProvider]() which provides validators
for attributes which derive from [System.ComponentModel.DataAnnotations.ValidationAttribute](). It also provides
a validator for types which implement [System.ComponentModel.DataAnnotations.IValidatableObject]().
