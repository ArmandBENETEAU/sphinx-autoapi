Summary
-------

A [Microsoft.AspNet.Mvc.Razor.IViewLocationExpander](Microsoft.AspNet.Mvc.Razor.IViewLocationExpander.yml) that adds the language as an extension prefix to view names. Language
that is getting added as extension prefix comes from @!:Microsoft.AspNet.Http.HttpContext-.
