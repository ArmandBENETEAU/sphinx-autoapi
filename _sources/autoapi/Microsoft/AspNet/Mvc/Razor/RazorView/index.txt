

RazorView Class
===============



.. contents:: 
   :local:



Summary
-------

Default implementation for [Microsoft.AspNet.Mvc.Rendering.IView]() that executes one or more [Microsoft.AspNet.Mvc.Razor.IRazorPage](Microsoft.AspNet.Mvc.Razor.IRazorPage.yml)
as parts of its exeuction.
