

RazorViewFactory Class
======================



.. contents:: 
   :local:



Summary
-------

Represents the default [Microsoft.AspNet.Mvc.Razor.IRazorViewFactory](Microsoft.AspNet.Mvc.Razor.IRazorViewFactory.yml) implementation that creates
[Microsoft.AspNet.Mvc.Razor.RazorView](Microsoft.AspNet.Mvc.Razor.RazorView.yml) instances with a given [Microsoft.AspNet.Mvc.Razor.IRazorPage](Microsoft.AspNet.Mvc.Razor.IRazorPage.yml).
