

AttributeRouteLinkGenerationEntry Class
=======================================



.. contents:: 
   :local:



Summary
-------

Used to build an [Microsoft.AspNet.Mvc.Routing.InnerAttributeRoute](Microsoft.AspNet.Mvc.Routing.InnerAttributeRoute.yml). Represents an individual URL-generating route that will be
aggregated into the [Microsoft.AspNet.Mvc.Routing.InnerAttributeRoute](Microsoft.AspNet.Mvc.Routing.InnerAttributeRoute.yml).
