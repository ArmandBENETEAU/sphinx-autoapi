

IActionSelectionDecisionTree Interface
======================================



.. contents:: 
   :local:



Summary
-------

A data structure that retrieves a list of [Microsoft.AspNet.Mvc.ActionDescriptor]() matches based on the values
supplied for the current request by @!:Microsoft.AspNet.Routing.RouteData.Values-.
