

EnvironmentTagHelper Class
==========================



.. contents:: 
   :local:



Summary
-------

@!:ITagHelper- implementation targeting &lt;environment&gt; elements that conditionally renders
content based on the current value of @!:IHostingEnvironment.EnvironmentName-.
