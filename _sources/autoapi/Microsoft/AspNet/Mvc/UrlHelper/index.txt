

UrlHelper Class
===============



.. contents:: 
   :local:



Summary
-------

An implementation of [Microsoft.AspNet.Mvc.IUrlHelper](Microsoft.AspNet.Mvc.IUrlHelper.yml) that contains methods to
build URLs for ASP.NET MVC within an application.
