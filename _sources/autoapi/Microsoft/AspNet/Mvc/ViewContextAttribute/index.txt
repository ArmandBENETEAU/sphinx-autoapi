

ViewContextAttribute Class
==========================



.. contents:: 
   :local:



Summary
-------

Specifies that a tag helper property should be set with the current
[Microsoft.AspNet.Mvc.ViewContext](Microsoft.AspNet.Mvc.ViewContext.yml) when creating the tag helper. The property must have a public
set method.
