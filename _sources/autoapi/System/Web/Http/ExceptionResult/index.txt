

ExceptionResult Class
=====================



.. contents:: 
   :local:



Summary
-------

An action result that returns a @!:StatusCodes.Status500InternalServerError- response and
performs content negotiation on an [System.Web.Http.HttpError](System.Web.Http.HttpError.yml) based on an [System.Web.Http.ExceptionResult.Exception](System.Web.Http.ExceptionResult.yml).
