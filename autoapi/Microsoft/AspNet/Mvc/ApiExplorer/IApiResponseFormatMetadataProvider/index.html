GetSupportedContentTypes(System.Type, System.Type, MediaTypeHeaderValue)¶

Gets a filtered list of content types which are supported by the [Microsoft.AspNet.Mvc.IOutputFormatter]() for the <paramref name=”declaredType” /> and <paramref name=”contentType” />.

Arguments:
  • declaredType (System.Type) – The declared type for which the supported content types are desired.
  • runtimeType (System.Type) – The runtime type for which the supported content types are desired.
  • contentType – The content type for which the supported content types are desired, or

<c>null</c> if any content type can be used.

type contentType:
 MediaTypeHeaderValue
rtype:System.Collections.Generic.IReadOnlyList{MediaTypeHeaderValue}
return:Content types which are supported by the [Microsoft.AspNet.Mvc.IOutputFormatter]().
