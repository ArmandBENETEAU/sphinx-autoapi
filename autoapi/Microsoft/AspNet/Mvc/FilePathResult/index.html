Summary¶

An [Microsoft.AspNet.Mvc.ActionResult]() that when executed will write a file from disk to the response using mechanisms provided by the host.

NormalizePath(System.String)¶

Creates a normalized representation of the given <paramref name=”path” />. The default implementation doesn’t support files with ‘’ in the file name and treats the ‘’ as a directory separator. The default implementation will convert all the ‘’ into ‘/’ and will remove leading ‘~’ characters.
