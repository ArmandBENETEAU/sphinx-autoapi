main¶

Command godep helps build packages reproducibly by fixing their dependencies.

Example Usage

Save currently-used dependencies to file Godeps:

$ godep save

Build project using saved dependencies:

$ godep go install

or

$ GOPATH=`godep path`:$GOPATH $ go install

Type¶

Command¶

A Command is an implementation of a godep command like godep save or godep go.

Dependency¶

A Dependency is a specific revision of a package.

Godeps¶

Godeps describes what a package needs to be rebuilt reproducibly. It’s the same information stored in file Godeps.
