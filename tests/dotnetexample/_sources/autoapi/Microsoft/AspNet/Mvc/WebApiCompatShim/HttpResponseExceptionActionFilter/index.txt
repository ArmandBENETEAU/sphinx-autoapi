

HttpResponseExceptionActionFilter Class
=======================================



.. contents:: 
   :local:



Summary
-------

An action filter which sets [Microsoft.AspNet.Mvc.ActionExecutedContext.Result]() to an [Microsoft.AspNet.Mvc.ObjectResult]()
if the exception type is [System.Web.Http.HttpResponseException](System.Web.Http.HttpResponseException.yml).
This filter runs immediately after the action.
